# 公考计划跟踪器

这是一个专为公务员考试备考设计的进度跟踪应用，帮助您管理120天的行测和申论学习计划。

## 功能特点

- **进度可视化**: 直观显示总体进度和每日计划
- **数据持久化**: 自动保存进度到本地存储，关闭浏览器后数据不会丢失
- **日历视图**: 以周为单位查看计划，清晰展示当前进度
- **互动操作**: 可标记完成、重置状态等
- **响应式设计**: 支持桌面端和移动端访问

## 如何使用

1. 将整个项目文件夹 (`gongkao_tracker`) 放在本地服务器上（如使用 Live Server 扩展或 Python 的 `python -m http.server`）
2. 或者，在支持本地文件访问本地存储的浏览器中直接打开 `index.html`

### 主要功能：
- 点击"标记为完成"按钮记录当日学习进度
- 使用日历视图快速跳转到任意日期
- 查看已完成天数、当前进度和剩余天数

## 技术细节

- HTML5 + CSS3 + JavaScript ES6
- 使用 localStorage 持久化存储进度数据
- 响应式布局，适配不同设备尺寸
- 渐变色彩搭配，现代界面设计

## 自定义

由于无法直接访问您的Word文档，当前计划是基于一般公考安排生成的示例。您可以根据自己的具体计划修改 `script.js` 中的 `generatePlanData()` 方法来适配实际学习内容。

## 浏览器兼容性

支持所有现代浏览器 (Chrome, Firefox, Safari, Edge)。
