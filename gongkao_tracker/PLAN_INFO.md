# 公考计划跟踪器 - 计划说明

## 计划概述

根据您提供的《公考计划.pdf》文档，我已经将120天的详细学习计划整合到应用中。该计划分为以下几个主要阶段：

### 第一阶段：方法学习（第1-30天）
- **目标**：从零基础开始，掌握行测和申论的基本方法
- **时间分配**：每天60分钟（40分钟行测 + 20分钟申论）
- **第1-10天**：资料分析入门 + 申论最基础
- **第11-20天**：判断推理入门 + 申论对策题
- **第21-30天**：言语理解入门 + 申论写作框架

### 第二阶段：刷题巩固（第31-60天）
- **目标**：通过大量练习巩固所学技巧
- **时间分配**：每天60分钟（50分钟刷题 + 10分钟订正）
- **第31-45天**：模块精刷，稳定正确率
- **第46-60天**：提速刷题 + 申论小题

### 第三阶段：真题精刷（第61-90天）
- **目标**：通过真题练习，适应考试难度
- **题源**：2018-2025年国考真题
- **重点**：行测拆分做，申论只做小题

### 第四阶段：省考适配（第91-110天）
- **目标**：适应四川省考特色
- **题源**：2018-2025年四川省考真题
- **特点**：行测100题、难度低于国考，申论偏基层/乡村/治理

### 第五阶段：冲刺复盘（第111天-考前）
- **目标**：通过复盘错题和背诵关键内容，稳固分数
- **时间分配**：每天1小时（刷旧错题 + 背公式口诀 + 记规范词模板）

## 使用说明

应用已自动根据您的详细计划更新了每一天的具体学习内容。您可以：

1. 每天打开应用查看当天的学习任务
2. 学习完成后点击"标记为完成"
3. 通过日历视图查看整个120天的进度
4. 随时查看总体完成百分比和剩余天数

## 里程碑节点

应用内置了几个关键里程碑：
- 第30天：完成方法学习阶段
- 第60天：完成模块刷题阶段
- 第90天：完成国考真题阶段
- 第110天：完成省考真题适配
- 第120天：完成全部备考计划
