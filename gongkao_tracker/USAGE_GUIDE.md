# 公考计划跟踪器使用指南

## 项目概述

这是一个专为公务员考试备考设计的进度跟踪应用，帮助您管理120天的行测和申论学习计划。应用具有直观的进度显示、数据持久化存储和响应式设计等特点。

## 功能特性

- **进度可视化**: 直观显示总体进度和每日计划
- **数据持久化**: 使用localStorage自动保存进度，关闭浏览器后数据不会丢失
- **日历视图**: 以周为单位查看计划，清晰展示当前进度
- **互动操作**: 可标记完成、重置状态等
- **响应式设计**: 支持桌面端和移动端访问

## 部署方式

### 方式一：使用Node.js服务器（推荐）

1. 确保系统已安装Node.js
2. 在项目根目录执行以下命令启动服务器：
   ```
   npm install
   npm start
   ```
3. 在浏览器中访问 `http://localhost:3000`

### 方式二：使用Python服务器

1. 在项目根目录执行以下命令启动服务器：
   ```
   python3 -m http.server 8000
   ```
2. 在浏览器中访问 `http://localhost:8000`

### 方式三：浏览器直接打开（部分功能受限）

对于支持本地文件访问localStorage的浏览器，可以直接双击打开`index.html`，但可能在某些浏览器中会因安全策略限制而无法保存数据。

## 自定义学习计划

目前应用使用默认的120天计划，您可以根据自己的实际情况修改学习计划：

### 修改配置文件

编辑`plan_config.json`文件，按以下格式自定义您的学习计划：

```json
{
  "totalDays": 120,
  "plan": {
    "1-20": {
      "title": "阶段标题",
      "description": "阶段描述",
      "subjects": ["主题1", "主题2", "..."]
    },
    ...
  },
  "milestones": [
    {"day": 20, "title": "里程碑标题"},
    ...
  ]
}
```

- `totalDays`: 总天数
- `plan`: 按时间段定义学习计划，格式为"开始天-结束天"
- `milestones`: 重要的里程碑节点

### 重新加载应用

修改配置文件后，刷新浏览器页面即可看到新的计划内容。

## 使用说明

### 开始使用

1. 启动服务器并打开应用
2. 应用将自动保存您的进度到本地浏览器存储
3. 每次访问都会恢复您上次的进度

### 日常操作

- **标记完成**: 点击"标记为完成"按钮记录当日学习进度
- **重置状态**: 点击"重置今日状态"取消当前天的完成标记
- **切换周次**: 使用"上一周"/"下一周"按钮浏览不同时间段的计划
- **跳转日期**: 点击日历视图中的某一天直接跳转到该天

### 进度跟踪

- 顶部进度条显示总体完成百分比
- 统计区域显示已完成天数、当前天数和剩余天数
- 日历视图通过颜色区分已完成、当前和待完成的日期

## 数据备份

重要进度数据存储在浏览器的localStorage中，为了防止数据丢失，建议：

1. 不要清除浏览器数据
2. 定期检查进度是否正常保存
3. 如需迁移数据，可通过浏览器开发者工具查看和导出localStorage内容

## 兼容性

支持所有现代浏览器：
- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## 故障排除

### 进度未保存
- 检查浏览器是否启用了隐私模式（隐私模式可能不保存localStorage）
- 确认浏览器支持localStorage功能
- 尝试刷新页面

### 无法加载配置文件
- 确认`plan_config.json`文件存在且格式正确
- 检查服务器是否正确提供了JSON文件

### 界面显示异常
- 清除浏览器缓存
- 检查CSS和JS文件是否正确加载
